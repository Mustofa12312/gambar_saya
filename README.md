# Sistem Keamanan Pelacak Perangkat

Aplikasi web sederhana untuk melacak lokasi perangkat dan mengirimkan laporannya secara otomatis ke email admin menggunakan EmailJS.

## Fitur

- Pelacakan Geolocation real-time.
- Pengiriman email otomatis (Latitude, Longitude, Google Maps Link).
- Desain UI Premium (Scanning Radar).
- Responsif untuk Mobile & Desktop.

## Teknologi

- HTML5 & CSS3
- JavaScript (Vanilla)
- [EmailJS SDK](https://www.emailjs.com/)
